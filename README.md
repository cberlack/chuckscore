# ChuckScore

Personal local-first tracker for "Chuck Score" — rate experiences with 1–5 stars across categories.

## Features

- React + Vite + TypeScript
- IndexedDB (Dexie) local storage
- Export / Import JSON
- Simple UI: Dashboard, Add Entry, Entries list, Details
- Deploys to GitHub Pages via GitHub Actions

## Quick start

1. npm install
2. npm run dev

## Build & deploy

The repository includes a GitHub Actions workflow that builds and deploys the `dist/` output to GitHub Pages.

## Notes

This is an MVP scaffold. Add images, maps, auth, or syncing later.
